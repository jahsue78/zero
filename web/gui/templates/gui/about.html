What is Zero ?

Shape is a modem trace management tool that allows modem experts and data scientists to collaborate on data projects that require a reliable and consistent modem trace data sets. It combines an efficient feature extraction tool that uses data flow caches and a user-friendly interface to handle data sets and explore meta-parameters.

What's Next ?

If you would like to contribute, please contact me. Among the next targets, the following ones have the priority:

  • Everything in browser: The goal of the project is to have all the shape manager features.
